							<h1>
							   Axis Misconfiguration: Debug Information
							</h1>
							<h2>ABSTRACT</h2>
							<p>
							  Debugging information helps attackers learn about the system and plan a form of attack.
							</p>
							<h2>EXPLANATION</h2>
							<p>
							  When <code>axis.development.system</code> is set to <code>true</code> in the server configuration file, the system behaves as if it is a development environment. It sends stack traces in SOAP responses that can leak information about the system or application.<br><br></p>
