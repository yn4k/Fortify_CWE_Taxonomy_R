							<h1>
							   Struts Misconfiguration: Missing Form Property Type
							</h1>
							<h2>ABSTRACT</h2>
							<p>
							  It is an error to define a <code>form-property</code> without a <code>type</code> type attribute.
							</p>
							<h2>EXPLANATION</h2>
							<p>
							  Struts requires <code>&lt;form-property&gt;</code> tags to include a <code>type</code> attribute. Struts will throw an exception when processing a form that defines a <code>form-property</code> with no type.<br><br><b>Example 1:</b> The following configuration omits a type for the <code>name</code> property.<br></p>
<pre><br>&lt;form-bean name="loginForm" type="org.apache.struts.validator.DynaValidatorForm"&gt;<br>  &lt;form-property name="name" /&gt;<br>  &lt;form-property name="password" type="java.lang.String" /&gt;<br>&lt;/form-bean&gt;<br></pre>
